VIAJA - Plataforma Inteligente de Viajes

Aplicación web tipo SPA (Single Page Application) enfocada en la planificación inteligente de viajes. Los usuarios pueden buscar y diseñar itinerarios usando filtros avanzados. El sistema orquesta una base de datos local sólida con la API de Inteligencia Artificial de Gemini, generando rutas personalizadas y estimaciones de costes en tiempo real bajo una arquitectura limpia y robusta.
